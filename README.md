# Cardápio Bella Pizza

Este projeto é um cardápio digital para a pizzaria Bella Pizza, desenvolvido utilizando HTML e CSS. Ele apresenta as opções de combos, pizzas, massas e bebidas disponíveis, com um design responsivo e otimizado para impressão.

## Estrutura do Projeto

- **index.html**: Página principal do cardápio, contendo as opções de combos e pizzas.
- **index_massas.html**: Página adicional com as opções de massas e bebidas.
- **style.css**: Arquivo de estilos que define o layout e a aparência do cardápio.
- **Logo.png**: Logotipo da pizzaria exibido no topo do cardápio.
- **Imagens adicionais**: Arquivos de imagem utilizados no projeto.

## Funcionalidades

- **Design Responsivo**: O layout é ajustado para diferentes tamanhos de tela.
- **Modo de Impressão**: Estilos específicos para impressão, garantindo que o cardápio seja bem apresentado no papel.
- **Organização Visual**: Uso de cores, fontes e espaçamento para destacar seções e informações importantes.

## Como Usar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em um navegador para visualizar o cardápio principal.
3. Para visualizar as opções de massas e bebidas, abra o arquivo `index_massas.html`.

## Personalização

- Para alterar os preços ou adicionar novos itens, edite os arquivos `index.html` e `index_massas.html`.
- Para modificar o design, edite o arquivo `style.css`.

## Pré-requisitos

- Um navegador moderno para visualizar o cardápio.
- Editor de texto (como Visual Studio Code) para personalizações.
